react usually doesn't need jquery

react manipulate the UI and jquery control UI. Contradict functioanlity . That's why this combo React + JQuery is not good


React does not need jQuery:
- DOM updates
- Events
- State & UI rendering
- Animations


JQuery's main job is to DOM manipulation, whereas react internally control itself the DOM. mixing both = so many bugs


When React + jQuery is used:
1. old code integration
2. use jQuery based plugins
3. Migration from old code to new code


JQuery is used with
1. HTML+ css + js
2. Boostrap
3. Server side framework (template based apps)
JQuery is heavily used with :
- Django (templates)
- Flask (jinja)
- PHP (laravel)
- ASP.NET
- Ruby on Rails (old apps)

4. CMS or website buiders (very common)
- wordpress
- drupal

5. Old front-end frameworks
- backbone js
- **angular js**
- Ember.js
